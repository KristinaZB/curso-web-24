Selectores css

Los selectores se usan para "encontrar" los elementos HTML a los elementos que queremos otorgar estilo

Tienen 5 categorias

  • Selectores simples: nombre, clase, id
  • Selectores combinadores: Seleccionan elementos basándosenen una relación específica entre ellos.
  • Selectores de Pseudo-clase: Seleccionan elementos basándose en el estado específico
  • Selectores de Pseudo-elementos: Seleccionan y estilizan ciertas partes de un elemento.
  • Selectores de atributo: Selecciona elementos basándose en un atributo o en el valor de un atributo.

Selector id

Selecciona un único elemento basándose en el atributo id

El atributo id es único por cada elemento. Si en el documento hay dos elementos con la misma id, ésta se aplicará solamente al último (orden de lectura)
