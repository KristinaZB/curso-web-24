Atributos en elementos de html

Todos los elementos pueden contener atributos. Los atributos proveen información adicional sobre los elementos.

Los atributos se especifican en la etiqueta de apertura. Se componen de dos partes, nombre y valor.

name="value"

href

Es el atributo básico de los links (a). Las (a) define un hipervínculo, el atributo href define la URL de la página donde nos va a llevar el hipervínculo. Página de busqueda

src

La etiqueta o elemento (img) se usa para insertar una imagen en una página HTML. El atributo src indica el camino de la imagen a mostrar.

src con URL o vínculo relativo.

width y height

Los atributos width, ancho y height, altura, definen el tamaño que queremos otorgar a la imagen en nuestra página.

alt

El atributo alt define el texto alternativo para una imagen. Este texto se mostrará si por alguna razón la imagen no puede cargarse.

style

Este atributo se sua para añadir estilos visuales a cualquier elemento. Como por ejemplo, color, fuente, -font-, tamaño, -size-, etc.

clic aquí para más detalles sobre el style

title

Este atributo se usa para proveer información extra sobre un atributo

Esta información se muestra en forma de tooltip (cuadrito de ayuda) al dejar el cursor sobre ese elemento
